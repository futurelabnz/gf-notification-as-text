Allows GF notifications to be sent as text format from the notifications settings page.

## Description

Enable this plugin to add a notification setting which will force email notifications to be sent as TEXT format instead of HTML.

Feel free to contribute on [github](https://github.com/futurelabnz/gf-notification-as-text).

## Frequently Asked Questions

### How does this work?

1. Enable plugin.
2. Navigate to GravityForms notification and edit.
3. Check the box titled "Send notification email as text format?"
4. Save notification settings.

That's it. This notification will now send as text, instead of HTML.
